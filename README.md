# Resume_Website
This is a personal bio-data website created using HTML. The website presents my personal details, educational background, skills, and contact information in a clean and organized format. It is designed to be simple, responsive, and easy to navigate, making it ideal for showcasing my profile online.

**Technologies Used**

HTML: To structure the web pages and display content.

CSS (optional): For styling and making the website visually appealing.

**Purpose**

The main purpose of this project is to demonstrate my web development skills and create a professional online bio-data that can be shared with recruiters or displayed on GitHub.

**Features**

Displays personal information clearly and neatly.

Lists educational qualifications and skills.

Provides contact details for easy communication.

Simple layout suitable for beginners to understand basic HTML structure.

Fully responsive and easy to navigate.

**GitHub Use**

This project can be hosted on GitHub Pages to make it accessible online. It serves as a portfolio project to showcase my foundational web development skills.
